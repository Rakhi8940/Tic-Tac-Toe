**Tic-Tac-Toe Web Game**
This is a classic Tic-Tac-Toe game built with HTML, CSS, and JavaScript. It allows two players to play the game, alternates turns between Player X and Player O, displays the game result (win or draw), and provides a reset feature for starting a new game. The game is fully responsive and optimized for different screen sizes.

**Features**
1. Player Turns: Alternates between Player X and Player O.
2. Game Result: Displays a win message when a player wins or a draw message when no one wins.
3. Reset Game: Allows resetting the game to start a new round.
4. Responsive Design: The layout adjusts for mobile, tablet, and desktop devices.
5. Interactive Cells: Players can click on a cell to place their symbol (X or O).
6. Color Styling: Cells are color-coded for each player (X in red, O in blue).

**Technologies Used**
1. HTML: Provides the structure and layout for the Tic-Tac-Toe board and buttons.
2. CSS: Styles the game board, cells, and responsive design.
3. JavaScript: Handles game logic, player turns, win checks, and game reset functionality.

**How to Play**
1. Start the Game:
  The game starts with Player X.
2. Take Turns:
  Click on any empty cell to place your symbol (X or O).
  The game alternates between Player X and Player O.
3. Win or Draw:
  The game will check for a winner after each turn. A message will display when one player wins, or if the game ends in a draw.
4. Reset the Game:
  Click on the "Reset Game" button to start a new round without reloading the page.
5. New Game:
  After a game ends, click the "New Game" button to reset and start a fresh game.
