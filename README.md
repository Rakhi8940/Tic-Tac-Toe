# Tic Tac Toe Game

A simple web-based Tic Tac Toe game built using **HTML**, **CSS**, and **JavaScript**. This project allows two players to play the classic Tic Tac Toe game in their web browser.

## Features

- Two-player gameplay: Player 1 (X) vs Player 2 (O)
- A grid layout with 3x3 cells
- Visual indication of the current player's turn
- Game over detection with a message showing the winner or a draw
- Restart functionality to play again
- Simple and responsive design

## Technologies Used

- **HTML**: Used to create the structure of the Tic Tac Toe grid and game interface.
- **CSS**: Used for styling the layout, grid, and buttons for a clean and modern look.
- **JavaScript**: Used to manage game logic, player turns, win conditions, and interaction handling.
